# 🎤 Voice Interview Simulator

Practice answering interview questions in a more natural way — by **speaking instead of typing**.

This project is a lightweight web application that simulates a basic interview experience where users respond to questions using their **voice**. The application listens to responses and analyzes them based on key concepts, creating an interactive way to practice technical thinking.

---

## ✨ Highlights

* 🎙 Voice-based answers using browser speech recognition
* ❓ Interview-style question prompts
* 🧠 Keyword-based response evaluation
* ⚡ Fast development environment powered by Vite

---

## 🛠 Tech

JavaScript • HTML • CSS • Vite

---

## 💡 Idea

Interviews are rarely about typing answers — they’re about **explaining your thoughts clearly**.
This project explores how voice interaction can make interview practice more engaging and realistic.
